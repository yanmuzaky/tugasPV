# Deskripsi Proyek: Aplikasi Booking Futsal
Aplikasi ini dibuat menggunakan C# Windows Forms di Visual Studio 2022 untuk mempermudah pengguna dalam melakukan pemesanan lapangan futsal.

# Fitur Aplikasi:
- Tambah Booking → Pengguna bisa memasukkan Nama, memilih Lapangan, serta menentukan Tanggal & Jam booking.
- Simpan Data ke CSV → Setiap booking yang dibuat akan disimpan dalam file bookings.csv agar bisa dibuka kembali.
- Tampilkan Data Booking → Data yang sudah tersimpan akan otomatis muncul di DataGridView saat aplikasi dibuka.

# Teknologi yang Digunakan:
- Bahasa: C#
- Framework: .NET Windows Forms
- Database: CSV (tanpa SQL)
- IDE: Visual Studio 2022

Aplikasi ini mempermudah pengelolaan booking lapangan futsal tanpa memerlukan database!
